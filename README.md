FastAdmin<sup>HTML</sup>是一款基于Bootstrap的极速后台开发模板。
===============


## **主要特性**

* 基于`AdminLTE`二次开发
* 基于`Bootstrap`开发，自适应手机、平板、PC
* 基于`RequireJS`进行JS模块管理，按需加载
* 基于`Less`进行样式开发
* 基于`Bower`进行前端组件包管理
* 基于第三方插件的增删改查一站式整合方案
* 一键压缩打包JS和CSS文件
* 多语言客户端支持
* 无缝整合又拍云上传功能
